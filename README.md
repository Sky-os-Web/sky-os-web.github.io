# Sky OS Website

GitHub Pages-ready website for Sky OS.

Upload the contents of this folder to the repository root, not the folder itself. Keep `assets/`, `docs/`, `.nojekyll`, and all HTML files together.

Main pages:

- `index.html`
- `experience.html`
- `design.html`
- `roadmap.html`
- `alpha.html`
- `technology.html`
- `compatibility.html`
- `docs.html`
- `download.html`
- `release.html`
- `community.html`
