Design direction

Quiet visuals. Clear places. No clutter.

Sky OS design is inspired by premium product simplicity, not by copying any one website or desktop. The interface should feel spacious, readable, and calm.

Identity

Space, calm, and focus.

The logo and wallpaper create a simple visual world: dark sky, soft light, and a planet mark that can work across boot screens, desktop, website, and community pages.

User interface goals.

The current desktop screenshot is early, but it gives Sky OS its own personality. The next design pass will round off rough edges, improve typography, add better spacing, and make the desktop feel less like a test screen.

Readable

Large text, simple panels, clear controls, and fewer technical labels on user-facing screens.

Consistent

One visual language across launcher, files, settings, system cards, and boot screens.

Honest

Preview builds should look polished without hiding what is still experimental.
