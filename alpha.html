Alpha release

Mid‑2027 is the target.

As of 17 June 2026, Sky OS is pre‑alpha. The alpha release is planned as the first public build meant for curious users to boot, explore, and test safely.

Preview statusCommunity updates

What alpha should include

Reliable boot into Sky OS, a desktop preview, file browsing, starter apps, diagnostics, and clear known limitations.

What alpha will not claim

It will not pretend to be a finished replacement for Windows, Linux, or macOS. Compatibility layers will stay clearly marked as experimental.

Who it is for

Early testers, OS-building learners, and people who want to watch a new desktop OS grow from the ground up.

Release quality matters.

Before any public image, Sky OS should boot repeatedly, open the shell and desktop safely, run simple apps, show clear errors, and keep serial logging available.
