View design direction
01

Not a remix.

Sky OS is not a reskinned Linux desktop. It boots from its own disk image, uses a custom stage‑1 and stage‑2 boot path, and runs a protected‑mode kernel.

02

Light by design.

The goal is a desktop that feels fast, calm, and easy to understand from first boot.

03

Open roadmap.

Preview releases will show the exact state of the OS without pretending unfinished systems are complete.

04

Built to grow.

The roadmap moves from boot reliability to memory management, programs, files, graphics, and finally a complete Sky Desktop experience.

Release window

Alpha in mid‑2027.

Current date reference: 17 June 2026. Sky OS is in active pre‑alpha development. The mid‑2027 alpha target focuses on a stable boot path, usable desktop preview, files, starter apps, and safe diagnostics.
