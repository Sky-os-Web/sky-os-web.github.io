Roadmap

From first boot to Sky Desktop.

The roadmap is written for users first: what each phase means, why it matters, and how it gets Sky OS closer to a real daily desktop.

V0.1

First heartbeat

A handmade boot sector and a tiny bootable preview prove the machine can start Sky OS.

V0.2

Kernel basics

Protected mode, VGA console, interrupts, timer, keyboard input, and early hardware control.

V0.4–V0.7

Debug, memory, programs, files

Serial logs, panic screens, memory maps, SkyFS, process structures, app experiments, and file commands.

V0.8

Graphics

Framebuffer graphics, mouse, font rendering, compositor experiments, and a smoother desktop preview.

V1.0

Sky Desktop

Graphical shell, windows, launcher, settings, file manager, and basic app toolkit.

Updates, screenshots, experiments, and community feedback live on Reddit.
